# Setup

These pages take you from a fresh SDK installation to a working conversational AI character. Follow them in order — each step builds on the previous one.

1. [Configure API Key](/broken/pages/66e922d1965681e0b9054a2fccf1fa17cb61897c) — connect your project to Convai
2. [Import and Run Sample Scenes](/broken/pages/06ee07b819d330074ce9db55c98b7a0b728222e0) — verify the SDK works before building your own scene
3. [Understanding Scene Components](/broken/pages/0b3783454f16e1f9bf03dc0754afdfcc1bad5cc3) — learn what each component does
4. [Custom Scene Setup](/broken/pages/23b06262705a3e1d681236e25a1b496bbdf52bd6) — build a scene from scratch
5. [Configure Conversation Input Mode](/broken/pages/d6ffc1495b2da554cb348868ef3a84862f8e4655) — choose hands-free or push-to-talk
6. [Configure Audio](/broken/pages/20d34ebf41b92e628a22d31af08d0b50c5e8bbbf) — tune microphone and playback settings
7. [Add Chat UI](/broken/pages/458fa2ebc261374a214e234660f4ada7f3c2b9be) — display conversation transcripts
8. [Add Lip Sync](/broken/pages/de1f7ac5149ad2cfd6fff1190deaf309d908b928) — drive character blendshapes from audio
9. [Validate Your Setup](/broken/pages/e7d75d481a373f3ae9acb4163d43f0e402a41be7) — confirm everything is working
