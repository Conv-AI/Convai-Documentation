# convai unity sdk

The Convai Unity SDK connects your Unity characters to Convai. Players speak; characters listen, understand, and respond — with synchronized lip sync, facial emotion, structured in-scene actions, and long-term memory across sessions.

{% hint style="success" %}
New to the SDK? Start with **Getting Started** — install the package and set up your first character.
{% endhint %}
