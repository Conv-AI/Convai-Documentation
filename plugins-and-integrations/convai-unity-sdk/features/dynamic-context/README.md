Dynamic Context gives Convai characters live awareness of the scene during a session. Use it for values that change after connection, such as a trainee's location, active hazards, collected equipment, recent interactions, or the object the character should focus on.

Runtime updates flow through `ConvaiCharacter.DynamicContext`. Scene and `UnityEvent` integrations can call that same surface through `ConvaiDynamicContextRelay`.
