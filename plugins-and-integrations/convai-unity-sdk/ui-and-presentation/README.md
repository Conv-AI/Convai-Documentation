# UI & Presentation

The Convai Unity SDK ships with a set of optional, scene-level UI components that you can drop into your project. None of these components communicate with Convai — they consume data already produced by the runtime and render it locally in your scene.

Each component is independent. Add only what your training simulation, interactive experience, or game actually needs.
