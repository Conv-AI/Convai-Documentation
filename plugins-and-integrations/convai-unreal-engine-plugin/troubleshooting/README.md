## How to gather logs

When an issue is hard to reproduce or unclear from the symptoms alone, enable verbose logging and collect the Output Log before filing a report.

See [Diagnostics and log export](diagnostics-and-log-export.md) for the full procedure, including which log categories to filter and how to save the output.
