# SDK Overview

### What the Convai Unity SDK Is and How It Works

The Convai Unity SDK layers voice, emotion, structured actions, and long-term memory onto Unity characters for training simulations, interactive experiences, and games. Each feature is a self-contained module — add only what your project needs, in the order that fits your workflow.
